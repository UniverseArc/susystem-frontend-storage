# Help Desk система
## Запуск
Клонируем репозиторий:
```
git clone git@gitlab.com:atremkoto/help-desk.git
```
## Без докера

Переходим в директорию проекта и ставим зависимости:
```
yarn
```
Запускаем dev-сервер и разрабатываем:
```
yarn dev
```
## С докером

Запустить Docker Desktop

```
docker compose up
