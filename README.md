# Help Desk система
## Запуск клиентской части для разработки

1) Клонируем репозиторий

## Без докера

2) Переходим в директорию проекта и ставим зависимости:
```
yarn
```
3) Запускаем dev-сервер и разрабатываем:
```
yarn dev
```
## С докером

1) Запустить Docker Desktop

```
docker compose up --build
```
